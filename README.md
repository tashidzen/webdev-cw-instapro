# Instapro

MVP социальной сети для обмена фотографиями
В проекте реализована следующая функциональность:
1) Вход и авторизация;
2) Добавление постов авторизованными пользователями;
3) Возможность оценки поста (лайк/дизлайк) авторизованными пользователями;
4) Удаление постов текущим пользователем;
5) Переход на общую страницу со всеми постами при нажатии на лого.
6) Переход на страницу постов конкретного пользователя при нажатии на имя пользователя в шапке поста.
7) Переход на список постов текущего пользователя при нажатии на "Моя страница".

## Технологии
В проекте использованы технологии: JavaScript, HTML, CSS (SСSS), БЭМ, Git, REST API.
